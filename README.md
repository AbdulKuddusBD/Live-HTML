# লাইভ HTML কোড প্র্যাকটিস ওয়েবসাইট

এটি একটি ফুল-স্ট্যাক ওয়েব অ্যাপ্লিকেশন যেখানে ইউজাররা সরাসরি HTML, CSS, এবং JavaScript কোড লিখে লাইভ ফলাফল দেখতে পারবেন।

## 🌟 ফিচারসমূহ

- ✨ **লাইভ প্রিভিউ** - কোড লেখার সাথে সাথে ফলাফল দেখুন
- 🎨 **সিনট্যাক্স হাইলাইটিং** - কোড আরও পড়তে সুবিধাজনক
- 💾 **স্নিপেট সংরক্ষণ** - কোড সংরক্ষণ করে রাখুন
- ⌨️ **কীবোর্ড শর্টকাট** - দ্রুত কাজ করার জন্য
- 📱 **রেসপন্সিভ ডিজাইন** - মোবাইল ও ট্যাবলেটে ব্যবহারযোগ্য
- 🔄 **অটো-রান** - স্বয়ংক্রিয়ভাবে কোড রান করুন

## 🚀 কিভাবে ব্যবহার করবেন

1. **HTML ট্যাবে** - HTML কোড লিখুন
2. **CSS ট্যাবে** - স্টাইলিং যোগ করুন
3. **JavaScript ট্যাবে** - ইন্টারঅ্যাকটিভিটি যোগ করুন
4. **রান বাটন** - কোড রান করতে ক্লিক করুন
5. **সেভ বাটন** - কোড সংরক্ষণ করতে ক্লিক করুন

## ⌨️ কীবোর্ড শর্টকাট

- `Ctrl/Cmd + Enter` - কোড রান করুন
- `Ctrl/Cmd + S` - স্নিপেট সংরক্ষণ করুন

## 🛠️ টেকনোলজিস

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Local Storage API

## 📦 ইনস্টলেশন

1. ফাইলগুলো ডাউনলোড করুন
2. আপনার ওয়েব সার্ভারে আপলোড করুন
3. `index.html` ওপেন করুন
